# AIStack Audit

AIStack Audit is a web application that helps users analyze AI subscription spending and discover potential savings opportunities.

## Current Features

- Dynamic AI spend form
- Savings audit engine
- AI-generated summaries
- Email capture
- Shareable audit URLs
- Express backend API

## Shareable Reports

Every saved audit generates a unique public URL that can be shared externally.

Example:
http://localhost:3000/audit/:id
