# Metrics

## Product Metrics

### Audits Generated
Measures total usage of the platform.

### Average Savings Identified
Tracks how much value the audit engine provides.

### Email Conversion Rate
Measures how many users submit email addresses after viewing audit value.

### Shareable Report Usage
Tracks viral sharing behavior and organic distribution.

# Technical Metrics

### API Response Time
Measures backend audit processing speed.

### Audit Generation Speed
Tracks frontend audit rendering performance.

### Public Audit Load Time
Measures dynamic route performance.

### Form Completion Rate
Tracks usability and onboarding effectiveness.

# Business Metrics

### User Retention
Percentage of returning users.

### Referral Traffic
Traffic generated from shared reports.

### Cost Optimization Accuracy
